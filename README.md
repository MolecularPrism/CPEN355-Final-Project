# CPEN355-Final-Project

## ⚙️ Setup (uv)

### 1. Create virtual environment
uv venv

### 2. Activate environment

macOS / Linux:
source .venv/bin/activate

Windows:
.venv\Scripts\activate

### 3. Install dependencies
uv pip install -r requirements.txt

### 4. Run the pipeline
python entry.py

### Verify environment (important)
which python

Expected:
.../.venv/bin/python
